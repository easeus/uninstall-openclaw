# uninstall-openclaw

A complete guide and scripts for uninstalling the [OpenClaw](https://github.com/openclaw/openclaw) personal AI assistant system.

---

## 📋 Installation Component Analysis

OpenClaw is a multi-component personal AI assistant system. After installation, it leaves the following traces on your system:

| Component | Location | Installation Method |
|---|---|---|
| **CLI Tool** | Global npm/pnpm/bun package | `npm install -g openclaw@latest` |
| **Gateway Daemon Service** | macOS: launchd / Linux: systemd / Windows: schtasks | `openclaw onboard --install-daemon` |
| **State + Config Directory** | `~/.openclaw/` (or `$OPENCLAW_STATE_DIR`) | Created automatically at runtime |
| **Config File** | `~/.openclaw/openclaw.json` (or `$OPENCLAW_CONFIG_PATH`) | Created by wizard/manually |
| **Workspace** | `~/.openclaw/workspace/` | Contains AGENTS.md, SOUL.md, skills, etc. |
| **Credentials** | `~/.openclaw/credentials/` | Created after channel login (e.g., WhatsApp) |
| **Log Files** | `/tmp/openclaw/openclaw-gateway.log` (macOS) | Generated by Gateway at runtime |
| **macOS App** | `/Applications/OpenClaw.app` | Optional installation |
| **macOS App LaunchAgent** | `~/Library/LaunchAgents/ai.openclaw.mac.plist` | Installed by macOS app |
| **Gateway LaunchAgent** | `~/Library/LaunchAgents/ai.openclaw.gateway.plist` | Installed by daemon install |
| **Swabble Voice Daemon** | `~/Library/LaunchAgents/com.swabble.agent.plist` | Optional (macOS 26) |
| **Swabble Config + Logs** | `~/.config/swabble/` and `~/Library/Application Support/swabble/` | Created by swabble setup |
| **Docker Containers/Images** | Docker runtime | Optional `docker-setup.sh` |
| **Podman Containers/systemd Quadlet** | `~/.config/containers/systemd/openclaw.container` | Optional `setup-podman.sh` |
| **Tailscale Serve/Funnel Config** | Tailscale system config | Optional |
| **Browser Profile** | Chrome/Chromium data managed by OpenClaw | Optional browser tools |
| **iOS/Android Device Pairing** | Device pairing info stored in Gateway | Optional pairing |
| **Environment Variables** | System shell profile | Set manually by user |
| **Nix Declarative Install** | home-manager/launchd service | Optional nix-openclaw |
| **Profile Variants** | `~/.openclaw-<profile>/` | Created when using `--profile` |

## 🚀 Quick Uninstall

### macOS / Linux

```bash
# Option 1: Run the uninstall script from this repo (recommended)
bash <(curl -fsSL https://raw.githubusercontent.com/easeus/uninstall-openclaw/main/uninstall.sh)

# Option 2: Clone and run
git clone https://github.com/easeus/uninstall-openclaw.git
cd uninstall-openclaw
chmod +x uninstall.sh
./uninstall.sh
```

### Windows

```powershell
# PowerShell (run as Administrator)
irm https://raw.githubusercontent.com/easeus/uninstall-openclaw/main/uninstall.ps1 | iex
```

## 🗑️ Complete Uninstall Plan

Follow this plan in order — **from high-level to low-level, from runtime to static files**.

### Phase 1: One-Click Auto Uninstall (if CLI is still available)

The simplest approach — try the built-in uninstaller first:

```bash
openclaw uninstall --all --yes --non-interactive
```

If the CLI has been removed but Node.js is still available, use npx:

```bash
npx -y openclaw uninstall --all --yes --non-interactive
```

> ⚠️ Even if the one-click uninstall succeeds, it is still recommended to continue with the remaining steps for final verification and cleanup of any residual files.

---

### Phase 2: Stop All Running Services

#### 2.1 Stop the Gateway Service

```bash
openclaw gateway stop
```

#### 2.2 Uninstall the Gateway Daemon

```bash
openclaw daemon uninstall
# or equivalently:
openclaw gateway uninstall
```

#### 2.3 Manual Stop (if CLI is unavailable)

**macOS (launchd):**
```bash
# Gateway service
launchctl bootout gui/$UID/ai.openclaw.gateway
rm -f ~/Library/LaunchAgents/ai.openclaw.gateway.plist

# If you have profile variants
# launchctl bootout gui/$UID/ai.openclaw.<profile>
# rm -f ~/Library/LaunchAgents/ai.openclaw.<profile>.plist

# Clean up possible legacy labels
ls ~/Library/LaunchAgents/com.openclaw.* 2>/dev/null && \
  launchctl bootout gui/$UID/com.openclaw.gateway 2>/dev/null; \
  rm -f ~/Library/LaunchAgents/com.openclaw.*.plist

# macOS app LaunchAgent
launchctl bootout gui/$UID/ai.openclaw.mac 2>/dev/null
rm -f ~/Library/LaunchAgents/ai.openclaw.mac.plist
```

**Linux (systemd):**
```bash
systemctl --user disable --now openclaw-gateway.service
rm -f ~/.config/systemd/user/openclaw-gateway.service
systemctl --user daemon-reload

# If you have profile variants
# systemctl --user disable --now openclaw-gateway-<profile>.service
# rm -f ~/.config/systemd/user/openclaw-gateway-<profile>.service
# systemctl --user daemon-reload
```

**Windows (Scheduled Tasks):**
```powershell
schtasks /Delete /F /TN "OpenClaw Gateway"
Remove-Item -Force "$env:USERPROFILE\.openclaw\gateway.cmd"
```

---

### Phase 3: Uninstall Optional Sub-Components

#### 3.1 Swabble Voice Daemon (macOS 26, if installed)

```bash
# Uninstall launchd service
launchctl bootout gui/$UID/com.swabble.agent 2>/dev/null
rm -f ~/Library/LaunchAgents/com.swabble.agent.plist

# Remove config and logs
rm -rf ~/.config/swabble/
rm -rf ~/Library/Application\ Support/swabble/
```

#### 3.2 Docker Installation (if docker-setup.sh was used)

```bash
# Stop and remove containers
docker compose down

# Remove Docker images
docker rmi openclaw:local 2>/dev/null
docker rmi openclaw-sandbox:bookworm-slim 2>/dev/null

# If a dedicated openclaw user was created
# sudo userdel openclaw (use with caution)
```

#### 3.3 Podman Installation (if setup-podman.sh was used)

```bash
# Stop the systemd quadlet service
systemctl --user disable --now openclaw.service 2>/dev/null

# Remove quadlet config
rm -f ~/.config/containers/systemd/openclaw.container
systemctl --user daemon-reload

# Remove Podman images
podman rmi openclaw:local 2>/dev/null
```

#### 3.4 Tailscale Configuration Revert (if enabled)

```bash
# If Serve or Funnel was enabled
tailscale serve off 2>/dev/null
tailscale funnel off 2>/dev/null
```

#### 3.5 macOS App

```bash
# Quit the app first
osascript -e 'quit app "OpenClaw"' 2>/dev/null
rm -rf /Applications/OpenClaw.app
```

---

### Phase 4: Remove Data and Configuration Files

```bash
# Main state directory (contains config, credentials, session data)
rm -rf "${OPENCLAW_STATE_DIR:-$HOME/.openclaw}"

# If you used a custom config path, remove it too
# rm -f "$OPENCLAW_CONFIG_PATH"

# If you used profiles, remove the corresponding state directories
# rm -rf ~/.openclaw-<profile>

# Log files (macOS)
rm -rf /tmp/openclaw/

# Workspace (if a separate path was configured)
# By default it is inside ~/.openclaw/workspace/ and was already removed above
```

---

### Phase 5: Uninstall the CLI Global Package

Choose the command matching your package manager:

```bash
npm rm -g openclaw
# or
pnpm remove -g openclaw
# or
bun remove -g openclaw
```

---

### Phase 6: Clean Up Source Installation (if installed via git clone)

```bash
# Remove the repository directory
rm -rf /path/to/openclaw/

# If pnpm was installed solely for openclaw
# npm rm -g pnpm
```

---

### Phase 7: Clean Up Environment Variables

Check and remove any manually set environment variables from your shell configuration files (`~/.bashrc`, `~/.zshrc`, `~/.profile`, etc.):

Variables to remove:
- `OPENCLAW_STATE_DIR`
- `OPENCLAW_CONFIG_PATH`
- `OPENCLAW_HOME`
- `OPENCLAW_PROFILE`
- `OPENCLAW_NIX_MODE`
- `OPENCLAW_GATEWAY_BIND`
- `OPENCLAW_GATEWAY_TOKEN`
- `OPENCLAW_SKIP_CANVAS_HOST`
- `TELEGRAM_BOT_TOKEN`
- `DISCORD_BOT_TOKEN`
- `SLACK_BOT_TOKEN`
- `SLACK_APP_TOKEN`
- `ZALO_BOT_TOKEN`
- Any other variables prefixed with `OPENCLAW_*`

macOS Nix mode defaults (if previously set):
```bash
defaults delete ai.openclaw.mac 2>/dev/null
```

---

### Phase 8: Verify Uninstallation

```bash
# Confirm CLI is removed
which openclaw 2>/dev/null && echo "⚠️ CLI still exists" || echo "✅ CLI removed"

# Confirm no residual processes
pgrep -f openclaw && echo "⚠️ openclaw processes still running" || echo "✅ No residual processes"

# Confirm no residual services (macOS)
launchctl list | grep -i openclaw && echo "⚠️ launchd services still exist" || echo "✅ No residual services"

# Confirm no residual services (Linux)
systemctl --user list-units | grep -i openclaw && echo "⚠️ systemd services still exist" || echo "✅ No residual services"

# Confirm directories are cleaned
ls -la ~/.openclaw 2>/dev/null && echo "⚠️ State directory still exists" || echo "✅ State directory cleaned"
```

## 📝 Notes

1. **Uninstall services before deleting files** — If running from source, always uninstall daemon services before deleting the repository. Otherwise, service configurations will point to non-existent paths.
2. **Multiple profiles must be cleaned individually** — If you used `--profile`/`OPENCLAW_PROFILE`, each profile has its own state directory and service instance.
3. **Remote Gateway** — If you run a Gateway on a remote Linux host, you must also perform the cleanup steps on that host.
4. **Third-party service integrations** — Telegram Bot, Discord Bot, Slack App, and other third-party configurations must be revoked/deleted manually on their respective platforms.
5. **Backup reminder** — Before uninstalling, consider backing up `~/.openclaw/workspace/` (contains your personalized AGENTS.md, SOUL.md, skills, etc.) and `~/.openclaw/openclaw.json` (configuration file).
